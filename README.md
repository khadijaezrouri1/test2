# Frutta Fresca 🍊

Sito e-commerce per la vendita di frutta fresca di stagione, consegnata direttamente a casa tua.

## Descrizione

Frutta Fresca è una piattaforma online che connette i clienti con i migliori produttori locali di frutta. Offriamo una selezione stagionale di prodotti freschi, con consegna rapida e packaging sostenibile.

## Funzionalità principali

- Catalogo frutta stagionale aggiornato settimanalmente
- Carrello e checkout sicuro
- Abbonamenti settimanali o mensili (cassette della frutta)
- Tracciamento ordini in tempo reale
- Filtri per origine, tipo e stagione

## Tecnologie utilizzate

- Frontend: Astro + Tailwind CSS
- Backend: Node.js
- Database: PostgreSQL
- Pagamenti: Stripe

## Avvio in locale

```bash
npm install
npm run dev
```

Il sito sarà disponibile su `http://localhost:4321`.
